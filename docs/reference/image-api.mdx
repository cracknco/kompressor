## `ImageCompressor`

```kotlin
public interface ImageCompressor {
    public suspend fun compress(
        inputPath: String,
        outputPath: String,
        config: ImageCompressionConfig = ImageCompressionConfig(),
    ): Result<CompressionResult>
}
```

<ParamField path="inputPath" type="String" required>
  Absolute filesystem path to the source image.
</ParamField>
<ParamField path="outputPath" type="String" required>
  Absolute filesystem path the compressed image will be written to. Overwritten if it exists.
</ParamField>
<ParamField path="config" type="ImageCompressionConfig" default="ImageCompressionConfig()">
  Format, quality, and resize parameters.
</ParamField>

<Note>
  No `onProgress` parameter — the underlying platform APIs (`Bitmap.compress`, `UIImageJPEGRepresentation`) are synchronous single-step operations.
</Note>

## `ImageCompressionConfig`

```kotlin
public data class ImageCompressionConfig(
    val format: ImageFormat = ImageFormat.JPEG,
    val quality: Int = 80,
    val maxWidth: Int? = null,
    val maxHeight: Int? = null,
    val keepAspectRatio: Boolean = true,
)
```

### Validation

Throws `IllegalArgumentException` at construction if:

- `quality` is outside `0..100`
- `maxWidth` is non-null and `≤ 0`
- `maxHeight` is non-null and `≤ 0`

## `ImageFormat`

```kotlin
public enum class ImageFormat {
    JPEG
}
```

Only `JPEG` is shipped today. PNG and WebP are planned.

## `ImagePresets`

public object ImagePresets {
